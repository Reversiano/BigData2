# Tutorial: Come scrivere un tutorial in Markdown

Markdown è un linguaggio di markup semplice e intuitivo, ideale per creare documentazione, tutorial e note.

## 1. Titoli

Usa il simbolo `#` per creare titoli:

```md
# Titolo principale
## Sottotitolo
### Sezione
```

## 2. Testo

Scrivi normalmente per il testo. Puoi **grassetto** con `**testo**` e *corsivo* con `*testo*`.

## 3. Liste

- Elemento 1
- Elemento 2
- Elemento 3

Oppure numerate:

1. Primo
2. Secondo
3. Terzo

## 4. Codice

Per inserire codice:

```python
print("Ciao mondo!")
```

## 5. Link e immagini

- [Link](https://www.example.com)
- ![Immagine](https://via.placeholder.com/150)

## 6. Tabelle

| Nome   | Età |
|--------|-----|
| Mario  | 25  |
| Lucia  | 30  |

## 9. Linea orizzontale

Usa `---`, `***` o `___` per una linea orizzontale.

---

## 10. Elenchi annidati

- Elemento 1
  - Sotto-elemento 1.1
  - Sotto-elemento 1.2
- Elemento 2

## 11. Blocchi di codice inline

Usa backtick singoli: `codice inline`

## 12. Escape di caratteri

Alguns caratteri speciali possono essere escapati con `\`:
\# \* \_ \[ \]
